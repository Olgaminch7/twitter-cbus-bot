# Twitter CBUS News Bot

Java appplication
Eclipse IDE
Deployed to Heroku

## Running Locally
Install the [Heroku CLI](https://cli.heroku.com/)

```sh
$ git clone https://github.com/heroku/java-getting-started.git
$ cd java-getting-started
$ mvn install
$ heroku local:start
```

## Deploying to Heroku

```sh
$ heroku create
$ git push heroku main
$ heroku open
```
